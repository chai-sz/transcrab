## #3：Security Unbroker

Unbroker 是一个 Hermes 原生技能，自动化了从网上删除你个人身份信息的整个流程。

它扫描数据中介注册表，识别你的数据出现在哪里，提交退出请求，验证删除结果，安排重新检查，并报告结果。

需要人工参与的部分——复杂的验证码、政府身份验证或电话确认——会被提升到人工任务队列。其余全部自动运行。

功能：

- 并行扫描数据中介和人员搜索网站
- 自动提交退出请求
- 优先处理集群父级删除
- 自动安排重新检查
- 生成完整的删除记录和审计报告

## 如何优化这些技能

安装技能只是第一步。

真正的价值来自于随着时间的推移不断优化，直到它完全匹配你的工作方式。这里是如何做到这一点（通用框架）：

**迭代循环**

每个技能都通过同一个循环变得更好：

运行 → 反馈 → 更新文件 → 再次运行

每当一个技能运行后输出不令人满意，你应该打开 SKILL.md 文件并添加一条规则。

示例

"Hermes，请更新我的品味技能，不要用蓝色色调——看起来像套话。"

我喜欢通过两种主要方式来优化我的技能：

1. **使用 /skill creator（Claude Code）**

这是改进现有技能的最快方法。在一次不满意的运行后打开 Claude Code 会话，输入：

/skill creator

然后，给它技能文件和你反馈。

2. **运行专门的优化循环**

对于你经常使用的技能，设置一个循环，根据你的反馈自动改进技能：

/loop 每次运行[技能名称]后，问我反馈，用我给出的新规则更新 SKILL.md，并保持对文件所有更改的变更日志。

这样，每次你使用该技能并指出问题时，改进就会被永久写入，而不是丢失。

注意：手动编辑技能文件或在 Cowork 等工具中附加它们也是好策略，这样可以随时轻松访问和更新它们。
